## 🛠️ How to Customize
1.  **Clone the repo:**
    ```bash
    git clone [https://github.com/diptarafd3t-cell/portfolio.git](https://github.com/diptarafd3t-cell/portfolio.git)
    ```
2.  **Update Images:** Replace the Imgur link in the `<img>` tag with your own profile photo.
3.  **Change Colors:** Modify the CSS variables in the `:root` section:
    ```css
    --accent-emerald: #10b981;
    --accent-purple: #9d50bb;
    ```
